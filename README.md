# PIGS-n-BLANKETS
Pigs-n-Blankets is an e-commerce boutique for pets. It features a product catalog, persistant user sessions & a checkout process.


![](pigs-n-blankets.gif)


## Getting Started
Clone or fork this repository

```
npm install
npm run dev-start
```

## Built With
Node, Express, React, Redux
